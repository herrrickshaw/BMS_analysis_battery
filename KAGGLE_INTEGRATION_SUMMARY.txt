STEP 3: MERGE DATASETS (30 mins)
────────────────────────────────

3.1 Standardize columns:
    python3 << 'PYTHON'
    import pandas as pd
    
    # Load all datasets
    ssri_df = pd.read_csv('ssri_outlets.csv')
    kaggle_df = pd.read_csv('kaggle_data.csv')
    
    # Standardize names
    ssri_df.rename(columns={
        'name': 'outlet_name',
        'latitude': 'lat',
        'longitude': 'lon'
    }, inplace=True)
    
    kaggle_df.rename(columns={
        'station_name': 'outlet_name',
        'lat_coordinate': 'lat',
        'lon_coordinate': 'lon'
    }, inplace=True)
    
    # Merge
    combined = pd.concat([ssri_df, kaggle_df], ignore_index=True)
    
    # Remove duplicates
    combined = combined.drop_duplicates(
        subset=['lat', 'lon'],
        keep='first'
    )
    
    print(f"Total records after merge: {len(combined)}")
    print(f"Duplicates removed: {len(ssri_df) + len(kaggle_df) - len(combined)}")
    
    # Export
    combined.to_csv('enhanced_outlets_kaggle.csv', index=False)
    PYTHON

3.2 Validate merged data:

STEP 4: CREATE ENHANCED MAP (20 mins)
─────────────────────────────────────

Use the provided Python script:
    python3 << 'PYTHON'
    import pandas as pd
    import folium
    from folium.plugins import MarkerCluster
    
    # Load merged data
    df = pd.read_csv('enhanced_outlets_kaggle.csv')
    
    # Create map
    center_lat = df['lat'].mean()
    center_lon = df['lon'].mean()
    
    m = folium.Map(
        location=[center_lat, center_lon],
        zoom_start=5,
        tiles='CartoDB positron'
    )
    
    # Add clustered markers
    cluster = MarkerCluster().add_to(m)
    
    for idx, row in df.iterrows():
        folium.CircleMarker(
            location=[row['lat'], row['lon']],
            radius=4,
            popup=f"{row['outlet_name']} - {row['state']}",
            color='blue',
            fill=True,
            fillOpacity=0.7
        ).add_to(cluster)
    
    # Save
    m.save('enhanced_india_outlets_kaggle.html')
    print("✓ Map saved: enhanced_india_outlets_kaggle.html")

⚠️ IMPORTANT NOTES
================================================================================

Coordinate Validation:
• Always validate lat/lon are within India bounds
• India: 8°N to 35°N latitude, 68°E to 97°E longitude
• Reject coordinates outside these bounds
• Remove null/zero coordinates

Duplicate Handling:
• Same lat/lon = likely duplicate
• Check within 0.01° tolerance (≈1 km)
• Keep first occurrence, remove duplicates
• Document which dataset was primary source

Data Quality:
• Verify at least 3 decimal places for lat/lon
• Check state matches coordinate region
• Compare against known reference points
• Document any discrepancies

Licensing:
• Check dataset license (prefer CC, Open)
• Document source for each record
• Add attribution column
• Follow Kaggle usage terms
