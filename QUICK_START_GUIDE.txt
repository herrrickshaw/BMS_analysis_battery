11. TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Problem: Pass rate too low (<3%)
Solution: Loosen profitability/growth thresholds
  • ROE: 15% → 12%
  • Earnings Growth: 12% → 10%
  • ROIC: 12% → 10%

Problem: Pass rate too high (>15%)
Solution: Tighten valuation thresholds
  • P/B: 1.0 → 0.8
  • P/E: 16 → 13
  • Add technical filter (above MA200)

Problem: Win rate declining (<45%)
Solution: Update fundamentals, check data freshness
  • Verify earnings <6 months old
  • Recalculate MA50/MA200/RSI
  • Review recent market conditions

Problem: High drawdown (>-8%)
Solution: Add defensive filters, reduce position size
  • Require Strong Liquidity >1.5x
  • Increase Interest Coverage >5.5x
  • Reduce position size 25%
