# Project Website

`index.html` is a self-contained interactive site documenting the Stock Analysis
System — features, the two analytical approaches, screener/RSI basis, and the full
version history (v1.0.0 → v3.12.0).

## View it
- **Locally:** open `docs/index.html` in any browser.
- **GitHub Pages:** enable Pages → Source: `main` branch, `/docs` folder.
  The site will be served at `https://herrrickshaw.github.io/herrrickshaw/`.

No build step, no dependencies — pure HTML/CSS/JS. Interactive: tabbed sections,
feature category filter, and a click-to-expand version timeline.

⚠️ Educational/research only. NOT investment advice.
