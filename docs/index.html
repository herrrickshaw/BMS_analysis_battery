⚖️The Two Approaches

Every stock is judged through two lenses with different failure modes. Their agreement is a cross-check — never a buy signal.

📊 Fundamental / Historical offline

Q: "Sound by its own numbers?"
Data: 5-yr OHLC + financials (Parquet).
Nature: deterministic, backtestable.
Output: "Stock Picks Based on Fundamentals".
Limit: backward-looking; misses catalysts.

🗞️ News / Sentiment online

Q: "What's the market saying now?"
Data: live headlines, company-name matched.
Nature: forward-looking, noisy.
Output: "Talk on the Street" + market mood.
Limit: provider-dependent; can mis-sign.

⭐Convergence

✅ Both bullish = highest conviction · ⚠️ strong fundamentals but negative news = caution. Leads the daily email.

▶️Run It

Self-contained Docker image; data cached as Parquet.

# build
docker build -t stockscan:3.12.0 .

# daily combined report (fundamentals + street talk + convergence)
docker run --rm -v $(pwd)/out:/app/output stockscan:3.12.0 report

# portfolio with a 20% yield target
python3 portfolio_builder.py --market IN --target-return 0.20 --max-weight 0.15

# news sentiment (IN: Moneycontrol/ET · US: CNBC/MarketWatch)
python3 pipeline_news.py --watchlist nifty50 --market IN
