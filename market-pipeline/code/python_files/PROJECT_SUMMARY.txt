   - ROA analysis (profitability)

   - Current ratio (liquidity)
   - Gross margin & asset turnover (efficiency)
   - Interpretation: Strong (≥7) / Moderate (4–6) / Weak (≤3)

   - PEG Ratio + Dividend Yield adjustment
   - Higher = Better value for dividend stocks
   - Customized for income-focused screening

5. Breakout Opportunities ✅
   - Distance from 200-day MA
   - Trend signal (Above/Below/Near)
   - Entry timing indicator

6. Dividend Yield ✅
   - Annual dividend as percentage
   - Recent dividend history

   - P/E (Price-to-Earnings)

================================================================================
                           TROUBLESHOOTING
================================================================================

Issue: "ModuleNotFoundError: No module named 'yfinance'"
Solution:
  $ pip install --upgrade yfinance pandas openpyxl

Issue: "No data found, symbol may be delisted"
Solution:
  Some older stocks unavailable on yfinance. Script skips them.
  Check SGX website for active listings.

Issue: "Too many requests" / Rate limiting
Solution:
  yfinance has ~2000 requests/hour limit.
  Reduce batch size: --sti-top15 instead of --sti30

Issue: Dividend Yield > 100%
Solution:
  yfinance data issue (annual vs quarterly mismatch).
  Check "Corporate Actions" section for actual recent dividends.

Issue: Script slow on batch run
Solution:
  Normal: 3–5 min for 15 stocks. Expected delays:
  - First run: yfinance data download
  - Network latency: ~2–3 sec per stock
  - Run during off-hours for faster response
