## 🔄 QUARTERLY REBALANCING

### Process
```
Every Quarter (Jan, Apr, Jul, Oct):

1. Data Collection (4h)
   ├─ Download OHLCV for all 250 stocks
   ├─ Fetch financial statements
   └─ Compile insider trading data

2. Scoring (6h)
   ├─ Recalculate Modern Resilience score for universe
   ├─ Rank all stocks by r_modern
   └─ Identify new top 50 per market

3. Execution (2-3h)
   ├─ Generate sell orders for dropped positions (avg 7-10 per market)
   ├─ Generate buy orders for new positions (avg 7-10 per market)
   ├─ Execute with limit orders (max 0.5% slippage)
   └─ Rebalance to target weights

4. Monitoring (1h)
   ├─ Verify all positions executed
   ├─ Confirm allocation matches targets
   ├─ Record trades for tax purposes
   └─ Update portfolio tracking
```
